This is a minimal Game that was made via TDGE Tower Defense Game Engine. The goal was to showcase the Core Game Loop, the Shop Functionality and provide for a short but fun experience.\hypertarget{md__c_1_2_users_2overo_2_desktop_2_game_lab_i_i_i-_t_d_g_e_2tdge_2docs_2wiki_2game_autotoc_md12}{}\doxysection{\texorpdfstring{How to start\+:}{How to start\+:}}\label{md__c_1_2_users_2overo_2_desktop_2_game_lab_i_i_i-_t_d_g_e_2tdge_2docs_2wiki_2game_autotoc_md12}
Follow the steps in "{}\+Quick\+Start.\+md"{}, then run the Example\+Game application.\hypertarget{md__c_1_2_users_2overo_2_desktop_2_game_lab_i_i_i-_t_d_g_e_2tdge_2docs_2wiki_2game_autotoc_md13}{}\doxysection{\texorpdfstring{How to play\+:}{How to play\+:}}\label{md__c_1_2_users_2overo_2_desktop_2_game_lab_i_i_i-_t_d_g_e_2tdge_2docs_2wiki_2game_autotoc_md13}
Just run the application and defend the End of the path by placing Turrets.

You can select which turret you want to place by clicking into the Shop windows.

Place the selected Turret by pressing \textquotesingle{}\textquotesingle{}T"{}.

The goal is to not let any enemies reach the end of the \doxylink{class_path}{Path}. Good Luck!
